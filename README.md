# Deus Adventures Auth인증

## 목표
- 실제 web 제작 전 세팅과 auth기능 구조잡기
- next.js 초기 세팅
- mobx 적용
- firebase auth를 이용하여 email, social Login 구현
- 익명사용자 회원가입 사용자와 연동
- 비밀번호 찾기

## 사용기술
- next.js
- mobx
- styled-components
- firebase

## 기능

- email / facebook / google Login
- 익명사용자 uid와 회원가입 사용자 uid 연동
- 비밀번호 재설정 email 전송
- next-i18next 언어 전환
